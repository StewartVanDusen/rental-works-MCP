**Step 4: Rewrite vitest.config.ts**

Replace the entire file with:

```typescript
import { defineConfig } from "vitest/config";

export default defineConfig({
  test: {
    environment: "node",
    projects: [
      {
        extends: true,
        test: {
          name: "unit",
          include: ["src/__tests__/unit/**/*.test.ts"],

});
```

CRITICAL: Remove `root: "src"` — it conflicts with repo-root-relative `include` paths in the projects array (see RESEARCH.md Pitfall 1). The `extends: true` on each project inherits the root-level `environment: "node"` setting.
